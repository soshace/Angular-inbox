Test task from Gwapit.

##How to deploy:
###**Manually**
1. **git clone** repository
2. open **app** directory
3. **npm install**
4. **node build.js**
5. **node index.js** - for production version
6. **node index.js debug** - for debug version

###**With docker**
1. **git clone** repository
2. **docker-compose build**
3. **docker-compose up**
